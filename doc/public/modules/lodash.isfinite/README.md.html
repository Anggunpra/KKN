lodash.isfinite v3.3.2

The lodash method _.isFinite exported as a Node.js module.

Installation

Using npm:

$ {sudo -H} npm i -g npm
$ npm i --save lodash.isfinite

In Node.js:

var isFinite = require('lodash.isfinite');

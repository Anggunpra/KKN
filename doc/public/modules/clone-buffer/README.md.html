clone-buffer

Easier Buffer cloning in node.

Example 

var cloneBuffer = require('clone-buffer');

var buffer = new Buffer('test');
var cloned = cloneBuffer(buffer);
// buffer !== cloned

API

cloneBuffer(buffer)

Takes a Buffer object and returns a clone. Throws if a non-Buffer is passed.

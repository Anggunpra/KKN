ext-list

Return a list of known file extensions and their MIME types

Install 

$ npm install --save ext-list

Usage 

const extList = require('ext-list');

extList();
//=> {'123': 'application/vnd.lotus-1-2-3', ez: 'application/andrew-inset', aw: 'application/applixware', ...}

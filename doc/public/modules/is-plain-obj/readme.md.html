is-plain-obj

Check if a value is a plain object

An object is plain if it's created by either {}, new Object() or Object.create(null).

Install 

$ npm install --save is-plain-obj

Usage 

var isPlainObj = require('is-plain-obj');

isPlainObj({foo: 'bar'});
//=> true

isPlainObj([1, 2, 3]);
//=> false

    <h1>Upload Preview jQuery Plugin</h1>

    <h2>What is that?</h2>
    <p>This jQuery plugin provides an easy way to preview your uploads before they're actually uploaded to the server. So if you selected an image or audio file from your hard drive, it will generate a live preview of the selected image or audio player for the audio file.</p>

    <hr />

    <h2>How it works?</h2>
    <p>To get access to the not uploaded data, we can use the HTML5 file reader api. This api provides reading local files. This step is pretty important, because we need to this data in order to show it in the browser window. To get more information about the file reader, you can read the <a href="http://www.w3.org/TR/FileAPI/" target="_blank">offical documentation</a>.</p>

    <hr />

    <h2>Download</h2>
    <p><a href="assets/js/jquery.uploadPreview.js">jquery.uploadPreview.js (2.5 kb)</a></p>
    <p><a href="assets/js/jquery.uploadPreview.min.js">jquery.uploadPreview.min.js (1.3 kb)</a></p>

    <hr />

    <h2>Installation</h2>
    <p><strong>1. Copy jQuery &amp; plugin:</strong></p>
<pre>
&lt;script type="text/javascript" src="http://code.jquery.com/jquery-2.0.3.min.js"&gt;&lt;/script&gt;
&lt;script type="text/javascript" src="assets/js/jquery.uploadPreview.min.js"&gt;&lt;/script&gt;
</pre>

    <p><strong>2. Set up your upload form e.g.:</strong></p>
<pre>
&lt;div id="image-preview"&gt;
  &lt;label for="image-upload" id="image-label"&gt;Choose File&lt;/label&gt;
  &lt;input type="file" name="image" id="image-upload" /&gt;
&lt;/div&gt;
</pre>

    <p><strong>3. Load the plugin function as often as you need and fill it with the parameters:</strong></p>
<pre>
&lt;script type="text/javascript"&gt;
$(document).ready(function() {
  $.uploadPreview({
    input_field: "#image-upload",   // Default: .image-upload
    preview_box: "#image-preview",  // Default: .image-preview
    label_field: "#image-label",    // Default: .image-label
    label_default: "Choose File",   // Default: Choose File
    label_selected: "Change File",  // Default: Change File
    no_label: false                 // Default: false
  });
});
&lt;/script&gt;
</pre>

    <hr />

    <h2>Customization</h2>
    <p>You're free to customize the CSS and HTML for the input, preview &amp; label fields. Feel free to change it as you wish. There's no need that you put it all together as you can see in step 2. The preview box might be located somewhere else on your page as the input field is.</p>
